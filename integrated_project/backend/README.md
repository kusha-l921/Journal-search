
# Integrated Project

## Backend Setup
```
cd backend
pip install -r requirements.txt
cp .env.example .env
# (edit .env if you change your API key later)
python app.py
```

## Frontend Setup
```
cd journal-muse-54-main
npm install
npm run dev
```

Then open http://localhost:5173 (React dev server).
The backend runs on http://localhost:5000
